features:
  - title: Multi-Agent System
    details: Mehrere KI-Agenten mit eigener Persoenlichkeit, Aufgaben und Gedaechtnis. Jeder Agent wird ueber einfache Markdown-Dateien konfiguriert — kein Code noetig.
  - title: Obsidian Vault als Brain
    details: Alle Daten liegen als plain Markdown im Dateisystem. Notizen, Aufgaben, Termine, Projekte — menschlich lesbar, versionierbar, kein Vendor-Lock-in.
  - title: Self-hosted & DSGVO
    details: Jeder Kunde bekommt einen eigenen EU-Server mit lokalem LLM (Ollama). Keine Daten verlassen den Server. Volle DSGVO-Konformitaet.
  - title: Telegram als Interface
    details: Kein separates Frontend noetig. Alles laeuft ueber Telegram — Nachrichten schreiben, Befehle ausfuehren, proaktive Erinnerungen empfangen.
  - title: Proaktiver Heartbeat
    details: Der Agent prueft automatisch Termine und Aufgaben und meldet sich nur wenn es etwas Relevantes gibt. Konfigurierbar per Cron-Ausdruck in einer MD-Datei.
  - title: Vollstaendig anpassbar
    details: Charakter, Ton, Regeln, Tools, Erinnerungen — alles wird ueber 10 Markdown-Dateien pro Agent gesteuert. Aendern ohne Neustart, ohne Programmierung.
